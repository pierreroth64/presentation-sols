## Build

- Installer [NodeJS](https://nodejs.org/) (idéalement via [nvm](https://github.com/nvm-sh/nvm))
- `nvm use` (optionnel)
- `npm install`
- `npm run dev`

## Déploiement

Pour la production: `npm run build` puis déployer le contenu du répertoire `./dist/`

## PDF

`npm run export` pour générer une version `pdf`
